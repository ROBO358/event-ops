# Final confirmation

最後に、以下の項目を確認しましょう。

- [ ] 目的は明確か
- [ ] 行動指針は決定したか
- [ ] 内容は明確か
- [ ] 会場は決定したか
- [ ] 日程は決定したか
- [ ] 会場は確保できたか
- [ ] 予算は決定したか
- [ ] 予算の確保方法は決定したか
- [ ] 予算の確保方法は実行可能か
- [ ] 各担当者は決定したか

## Summary

[What you want to do](./want-to-do)の繰り返しになりますが、企画段階ではイベントの目的や内容を明確にすることが重要です。この段階では、ごく少人数で行うべきでしょう。企画段階で多くの人が参加していると、方向性や目的を定めるためのコミュニケーションコストが高くなり多くの時間を費やすことになります。
